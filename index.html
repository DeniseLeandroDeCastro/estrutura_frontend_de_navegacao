O que é Algoritmo?

Já parou para pensar como um computador "entende" o que você quer que ele faça? O computador apenas segue as instruções que são passadas por você. Algoritmo é um conjunto de regras que fornecem uma sequência de operações capazes de resolver um problema específico. Uma receita de bolo é um algoritmo, por exemplo. Na receita, você tem os ingredientes e no algoritmo de um programa você terá as variáveis. Na receita de bolo você terá o passo-a-passo da maneira de fazer, assim também acontecerá no algoritmo de um programa de computador.

É preciso fazer exercícios constantemente. Seja disciplinado(a)!

O que é BackEnd?

É a parte do sistema responsável pelas regras de negócio, webservices e APIs. O Backend está por trás da aplicação, fazendo a ponte entre os dados que são enviados através do navegador rumo ao banco de dados e vice-versa, aplicando as regras de negócio.

O BackEnd é a parte que não está sendo vista, é no backend por exemplo que será definido como o servidor irá se comunicar com o banco de dados. Algumas linguagens de backend são:

  • PHP
  • Java
  • Python

O que é FrontEnd?

O frontend é o responsável por permitir que o usuário consiga visualizar, clicar ou ter algum tipo de interação com o sistema. O desenvolvedor frontend deve garantir que o conteúdo do site apareça corretamente em qualquer tamanho de tela. Todo detalhe é muito importante, pois o usuário precisa ter uma experiência que seja agradável, tanto no visual como na usabilidade.

Para iniciar como desenvolvedor frontend, é preciso aprender HTML, CSS e JavaScript.

Android

O Android é projetado principalmente para dispositivos móveis, com tela sensível ao toque para que o usuário possa manipular objetos virtuais, assim como um teclado virtual. O código do sistema operacional é disponibilizado pelo Google sob licença de código aberto. O Android é muito popular entre empresas de tecnologia que buscam um software pronto e personalizável para dispositivos de alta tecnologia.

O que é CSS?

O CSS é usado para estilizar elementos escritos em HTML, separa o conteúdo da representação visual do site. Com o CSS é possível alterar a cor do fundo da página, a cor do texto, o espaçamento entre parágrafos, também é possível criar tabelas, layouts, ajustar imagens para as telas, enfim, o CSS deixa o visual bem agradável.

O CSS é uma ferramenta potente que possibilita criar funcionalidades ao invés de usar JavaScript ou outra linguagem mais pesada. Usando moderadamente, pode viabilizar uma ótima experiência tanto ao desenvolvedor como aos usuários.
